# Obtiene un proveedor de producto por id
Obtiene el proveedor de productos en base al id dado.

### Ejemplo de Consulta
```
http://api.nyandelivery.io/proveedorProductos/11111
```

```
curl --request GET \
http://api.nyandelivery.io/proveedorProductos/11111
```

### Ejemplo de Respuesta

```
[
  "Mc Donald",
]
```
