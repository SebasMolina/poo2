# Eliminar un proveedor de productos
Elimina el proveedor de productos en base al id dado.

### Ejemplo de Consulta
```
http://api.nyandelivery.io/proveedorProductos/11111
```

```
curl --request DELETE \
http://api.nyandelivery.io/proveedorProductos/11111
```

### Ejemplo de Respuesta

```
200 OK 
```
